When you need to GET/POST data from your application and you like to do it asynchronously, well, do it in the right way!


Ajaxable is just a ~2.7KB (min + gzip) library which helps your forms degrade gracefully, maintaining progressive enhancement approach
For more info about Ajaxable and its usage check the repository on Github
