## Development

Clone the repository and enter inside the folder

```sh
$ npm i ajaxable
$ cd ajaxable
```

Install all necessary dependencies

```sh
$ npm i
```

Start dev server

```sh
$ npm start
```

Build before commit. This will also increment the patch level version of the package. (npm version patch)

```sh
$ npm run build
```
